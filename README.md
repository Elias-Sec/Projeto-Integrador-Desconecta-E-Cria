# 📦 Off-Box — Desconecte e Crie

O **Off-Box** (parte da iniciativa *Desconecta & Cria*) é uma plataforma web desenvolvida para ajudar famílias no "detox digital" infantil. O projeto oferece uma curadoria prática de brincadeiras e atividades manuais offline, incentivando o desenvolvimento saudável, a autonomia e a conexão familiar longe das telas.

---

## 🚀 Funcionalidades do Projeto

O site conta com um fluxo completo e responsivo, dividido nas seguintes páginas:

* **Página Inicial (`index.html`):** Apresenta a proposta de valor do projeto, os principais benefícios de reduzir o tempo de tela e chamadas para ação (CTAs) intuitivas.
* **Catálogo de Atividades (`Brincadeiras.html`):** Uma área interativa com filtros avançados de busca — por tempo de duração, tipo de atividade (sensorial, artesanal, etc.), materiais necessários e número de participantes — facilitando a escolha da dinâmica ideal.
* **Detalhes da Atividade (`oficina.html`):** Página dedicada a guiar a execução de uma brincadeira específica (como a *Oficina de Pintura com Dedos*), contendo tempo estimado, materiais necessários, guia passo a passo estruturado e um vídeo explicativo.
* **Planos de Assinatura (`planos.html`):** Tela que exibe as opções de planos (Gratuito, Premium e Anual) com tabelas comparativas de benefícios.
* **Área de Cadastro e Login (`Assinatura.html` e `login.html`):** Telas com formulários estruturados para novos usuários criarem suas contas ou acessarem a plataforma.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando práticas modernas de desenvolvimento web focado em responsividade e performance:

* **HTML5:** Estruturação semântica de todas as páginas.
* **Bootstrap 5:** Framework CSS utilizado para agilizar a estilização, garantir a responsividade (grid system) e fornecer componentes prontos.
* **CSS3 Customizado (`style.css`):** Folha de estilo personalizada para aplicar a identidade visual única da marca (cores institucionais, fontes e espaçamentos personalizados).
* **Vanilla JavaScript:** Para manipulação básica e inicialização de recursos interativos do Bootstrap.
